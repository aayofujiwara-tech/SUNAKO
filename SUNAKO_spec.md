## ゲームコンセプト

- 瞬間的なポーカー役の判断力と、ブラフ×スピードの駆け引きを楽しむ2人対戦ゲーム
- 「強い手を求めるか、スピードを取るか」のジレンマがコア
- 相手の宣言が脅威になって焦る設計（タイマーではなく相手起点のプレッシャー）

---

## ゲームルール

### 役（ポーカー標準）

ハイカード ＜ ワンペア ＜ ツーペア ＜ スリーオブアカインド ＜ ストレート ＜ フラッシュ ＜ フルハウス ＜ フォーオブアカインド ＜ ストレートフラッシュ ＜ ロイヤルフラッシュ

### 勝敗条件

- フォールド回数を使い切ると負け確定
- 先取ポイントに達したプレイヤーが勝利
- 役が同じ場合は引き分け（スコア移動なし）

### アクション

| アクション | 内容 |
|---|---|
| 宣言 | 現在の最強役で相手に勝負を挑む。ブラフ可。常に可能 |
| 受ける | 相手の宣言に応じる。双方の役を公開して比較、強い方が1点獲得 |
| フォールド | 相手の宣言を拒否。回数を使い切ると負け確定 |
| 交換 | 手札を引き直す。何度でも可能 |

### カウントダウン

- 相手が宣言した瞬間に発生
- 時間内に「受ける」か「フォールド」か選ぶ
- 時間切れは強制フォールド扱い

### ラウンドフロー

1. 手札配布
2. 引き直し・宣言・フォールドを繰り返す
3. 「受ける」が選択されたら役比べ
4. 勝敗決定 → 次のラウンドへ
5. 先取ポイントに達したプレイヤーが勝利

### ラウンド間のリセット

- 手札：毎ラウンドリセット
- フォールド回数：毎ラウンドリセット

---

## ゲームモード

### モードA：ドローポーカー型

- 手札7枚を一気に配布
- 引き直しは2枚丸ごと交換、何度でも可、常に宣言も可
- 7枚から最強の5枚を自動選択して役確定
- **緊張の源泉**：強欲に引き直すかスピードを取るか

### モードB：テキサスホールデム型

- 手札2枚配布
- 引き直しは2枚丸ごと交換、何度でも可、常に宣言も可
- コミュニティ5枚が時間経過で1枚ずつ自動めくれ
- 手札＋コミュニティから最強5枚で役確定
- カウントダウン中もコミュニティはめくれ続ける
- **緊張の源泉**：めくれるたびに役が変化、いつ宣言するかの連続判断

## UI仕様

### 全体レイアウト

- 上下分割（相手が上、自分が下）
- スマホ・PC両対応（レスポンシブ）
- カラーテーマ：カジノグリーン系（CSS変数で管理、後から変更可能）

### カードデザイン

- A / 2〜10 / J / Q / K ＋ マーク（♠♥♦♣）
- シンプルデザイン、視認性重視

### モードA レイアウト

```
[ 相手エリア：手札7枚横並び（裏面） ]
[ 中央エリア：スコア・フォールド残数 ]
[ 自分エリア：手札7枚横並び ]
[ プログレスバー ]
[ 交換 | 宣言/受ける | フォールド ]
```

### モードB レイアウト

```
[ 相手エリア：手札2枚（裏面） ]
[ 中央エリア：コミュニティ5枚 + スコア ]
[ 自分エリア：手札2枚 ]
[ プログレスバー ]
[ 交換 | 宣言/受ける | フォールド ]
```

### ボタン仕様

- 左：交換
- 中央：宣言（相手が宣言中は「受ける」に切り替わる）
- 右：フォールド
- ボタン上にプログレスバー表示

### 演出

| 演出 | 内容 |
|---|---|
| 役名表示 | 宣言時のみ表示（普段は非表示） |
| 相手の引き直し | シャッフルアニメーション |
| 役比べ | 双方手札公開 → 勝った役ハイライト → 数秒後次ラウンドへ |
